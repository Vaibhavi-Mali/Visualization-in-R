# Visualization-in-R

## R studio commands
1. How to clear R studio console
`CNTRL + L`

## Visualization Commands

1. How to load dataset
```
cricket <- read.csv("C:/Users/User/Downloads/cs.csv")
```

2. How to view data 
```
View(cricket)
```

3.	To get first 7 rows
```
head(cricket,7)
```

4. To remove last 7 rows and print remaining data 
```
head(cricket,-7)
```
